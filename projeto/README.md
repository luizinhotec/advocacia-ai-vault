# projeto/ — Gestão do Projeto

Esta pasta é reservada para **documentos de gestão do projeto** que não se encaixam nas camadas wiki/ ou raw/.

---

## O que entra aqui

- Propostas comerciais (versão sem dados financeiros sensíveis, ou referência à localização externa)
- Cronograma de entregas e marcos
- Atas de reunião de alinhamento *(sem dados de clientes)*
- Plano de onboarding do escritório
- Relatórios de progresso

---

## Diferença em relação às outras pastas

| Pasta | Conteúdo |
|-------|----------|
| `raw/interno/` | Documentos brutos que o agente LLM vai compilar em notas wiki |
| `wiki/interno/` | Conhecimento compilado — notas estruturadas com frontmatter |
| `projeto/` | Documentos de gestão — não precisam ser compilados, são lidos diretamente |
