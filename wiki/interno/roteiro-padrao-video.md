# Roteiro Padrão de Vídeo

Template obrigatório para todo vídeo produzido com o avatar. Garante conformidade OAB e consistência de marca.

---

## Estrutura obrigatória

Todo roteiro deve seguir esta sequência. Tempos são orientativos para vídeo de 60–90s.

```
┌─────────────────────────────────────────────────────────────┐
│  BLOCO 1 — ABERTURA (5–10s)                                 │
│  Identificação do avatar e do escritório.                   │
│  "Olá, eu sou [Nome do Avatar], IA do escritório [Nome]."  │
│  ⚠️ Obrigatório identificar como IA em todo vídeo novo.    │
├─────────────────────────────────────────────────────────────┤
│  BLOCO 2 — GANCHO (5–10s)                                   │
│  Pergunta ou situação do cotidiano que o espectador         │
│  reconhece. Nunca prometer resposta completa.               │
│  Ex: "Você sabe o que fazer quando..."                      │
├─────────────────────────────────────────────────────────────┤
│  BLOCO 3 — CONTEÚDO EDUCATIVO (30–60s)                      │
│  Explicação geral do tema — conceitos, contexto.            │
│  ❌ Proibido: "no seu caso você deve...", citar solução      │
│  concreta, prometer resultado.                              │
│  ✅ Permitido: "em geral, a lei prevê...", "é comum que..." │
├─────────────────────────────────────────────────────────────┤
│  BLOCO 4 — CTA (10–15s)                                     │
│  Redirecionamento para consulta ou para seguir o perfil.   │
│  ❌ Proibido: usar honorário/desconto como CTA.             │
│  ✅ Permitido: "agende uma consulta", "siga para mais       │
│  conteúdos", "mande uma mensagem para saber mais".         │
└─────────────────────────────────────────────────────────────┘
```

---

## Template de roteiro (copiar para cada vídeo novo)

```markdown
## Roteiro: [TÍTULO DO VÍDEO]
**Data de criação:** YYYY-MM-DD
**Pilar de conteúdo:** Educativo / Institucional / Informativo / Humanização
**Canal de destino:** Instagram Reels / LinkedIn / YouTube / Site / WhatsApp
**Duração estimada:** __s
**Aprovado por:** *(a preencher antes de produzir)*
**Data de aprovação:** *(a preencher)*

---

### BLOCO 1 — ABERTURA
[AVATAR]: *(texto do roteiro)*

### BLOCO 2 — GANCHO
[AVATAR]: *(texto do roteiro)*

### BLOCO 3 — CONTEÚDO EDUCATIVO
[AVATAR]: *(texto do roteiro)*

### BLOCO 4 — CTA
[AVATAR]: *(texto do roteiro)*

---

### Notas de produção
- Expressão facial/tom: *(ex: sorridente e confiante na abertura, sério e atento no conteúdo)*
- B-roll sugerido: *(imagens de apoio se houver)*
- Legenda/caption para publicação: *(texto da legenda do post)*
- Hashtags: *(a definir por canal)*
```

---

## Checklist antes de enviar para aprovação

- [ ] Bloco 1 identifica o avatar como IA?
- [ ] O vídeo faz alguma afirmação jurídica concreta? (se sim → revisar ou mover para raw)
- [ ] O CTA usa honorário como atrativo? (se sim → remover)
- [ ] O roteiro está alinhado com o pilar de conteúdo declarado?
- [ ] O tom é compatível com a [[identidade-do-avatar]]?
- [ ] A duração estimada está dentro do formato declarado?

---

## Exemplos de frases proibidas vs. permitidas

| ❌ Proibido | ✅ Alternativa permitida |
|------------|------------------------|
| "No seu caso, você tem direito a..." | "Em situações assim, a legislação prevê que..." |
| "Se você fizer isso, vai ganhar a causa" | "O ideal é buscar orientação jurídica para entender suas opções" |
| "Venha, primeira consulta gratuita!" | "Agende uma consulta e conheça suas alternativas" |
| "Somos os melhores em..." | "Nossa equipe atua há X anos em..." |
| "A lei X do artigo Y resolve o seu problema" | "A Lei X trata desse tema — um advogado pode analisar como ela se aplica ao seu caso" |
