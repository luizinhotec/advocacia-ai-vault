# Perguntas Frequentes

> **Nota para o bibliotecário:** Cada pergunta deve ter resposta informativa e direcionar para consulta.
> Proibido: responder com orientação jurídica concreta, citar artigos de lei como solução para o caso do cliente,
> afirmar o que "vai acontecer" no caso dele. Se uma resposta não puder ser dada sem orientação concreta, a resposta padrão é:
> *"Essa situação depende de detalhes específicos do seu caso. O ideal é agendar uma consulta para que possamos analisá-la com atenção."*
